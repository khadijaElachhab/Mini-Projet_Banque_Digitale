Identifiants de connexion

L'application est préconfigurée avec deux utilisateurs par défaut :

### Administrateur
- **Username** : `admin`
- **Password** : `admin123`
- **Rôles** : ADMIN, USER
- **Accès** : Toutes les fonctionnalités (gestion clients, comptes, opérations, dashboard)

### Utilisateur standard
- **Username** : `user`
- **Password** : `user123`
- **Rôles** : USER
- **Accès** : Consultation le dashboard , clients et comptes uniquement
